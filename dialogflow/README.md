# Agente Dialogflow - Digital Circus STEM Escape

Esta carpeta contiene el agente de Dialogflow desarrollado para la interacción conversacional de Caine.

El agente incluye:
- Bienvenida e inicio de la aventura.
- Cinco salas con desafíos educativos.
- Respuestas correctas y validaciones.
- Dos pistas progresivas por cada sala.
- Control de hasta tres intentos fallidos.
- Contextos para controlar el avance entre salas.
- Portal Final al completar las cinco llaves.
- Interacción mediante texto y reconocimiento de voz.

El archivo ZIP exportado desde Dialogflow se incluye como respaldo de la configuración del agente.
